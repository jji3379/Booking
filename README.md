# *도서 추천 웹 서비스 Booking* 
[www.acornsbook.me](http://3.23.177.146:8080/booking/home.do)

# 사용 기술 
- Java, Spring, Mybatis, Oracle DB
- Html, CSS, Javascript, JQuery, Ajax, BootStrap
- Apache Tomcat
 
# 사용한 Api

 ----------- | -------------- | -------------
